Traffic Analysis on Car Crashes

Dataset

This project is part of a personal project using data retrieved from internet, The dataset is available on Github.

Objective

The main goals in this project are to explore and analyze the data collected to uncover insights and correlations. In addition, to build a model that predicts car crashes based on the data provided.

Task

  1. Exploring the dataset.
  2. Data visualization to analyze correlations and perform descriptive statistics.
  3. Prepare a linear regression model.
  4. Compare results from model to actual.

Action

Coding, insights and conclusions are also available here

Results

  • There is a significant correlation between smartphone usage and the crash rate. This suggests that as smartphone usage increases, crash rates tend to rise as well.
  • However, the notebook notes that other factors beyond smartphone usage (e.g., seasonal effects or external factors) could also be influencing crash rates.
  • The increasing trend in smartphone usage might lead to more crashes over time, indicating a potential area for future policy or behavioral intervention.
