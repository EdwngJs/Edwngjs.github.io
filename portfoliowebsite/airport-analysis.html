Airport Analysis

Dataset

For this project, I used an airline database from Kaggle. The dataset includes multiple tables from a SQL database contains the aircraft and airport data, bookings, flights, boarding passes, etc. For this project, I used the tables that contain the flight, airport and ticket data.

Objective

The objective of this project was to utilize SQL and Python for data retrieval, transformation, cleaning, and analysis of flight data. The primary goals were to identify causes of flight delays and explore correlations, as well as to analyze the revenues generated from these flights.

Task

  1. Retrieve the flight data from the database using SQL queries.
  2. Load the retrieved data into Python as a pandas dataframe for data transformation and cleaning.
  3. Analyze the cleaned data to identify causes of flight delays and explore correlations between different variables.
  4. Analyze the revenue data associated with the flights.

Action

Data Retrieval: Utilized SQL queries to extract the flight data from the database, including information such as flights, departure and arrival times and airports, and ticket sales.

Data Transformation and Cleaning: Loaded the retrieved data into Python and performed necessary transformations and cleaning procedures to ensure data quality and consistency. This involves handling missing values, standardizing formats, perform required calculations and merging datasets.

Analysis for Flight Delays: Analyzed the cleaned data to identify patterns and factors contributing to flight delays. This involve exploring other variables such as airport outbounds and distance between airports

Correlation Analysis: Explored correlations between arrival and departure delays to understand relationships and potential factors influencing flight delays.

Revenue Analysis: Analyzed the revenue data associated with the flights to identify revenue trends and profitability of different routes and distances.

Results

  • Explore causes of flight delays.

    • Key Takeways / Lessons learned

  • Utilizing SQL and Python together allows for comprehensive data analysis and insights generation.

  • With this project I could perform a lot of data cleaning and manipulation to string data and also json objects.
  • Write functions to calculate statistics parameters and flight distances using the given coordinates.
  • If I were to work with this project I would explore other factors that could impact the flight delays and determine actions for the potential root causes.
