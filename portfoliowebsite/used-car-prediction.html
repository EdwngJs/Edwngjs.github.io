Used Car Price Prediction

Dataset

For this project, I used a dealer car database. The table includes multiple features related to cars.

Objective

Train 3 Machine Learning algorithms namely Multiple Linear Regression, Random Forest Regression, and XG-Boost to predict used cars prices. This project can be used by car dealerships to predict used car prices and understand the key factors that contribute to used car prices.

Task

  1. Cleaning and exploring the dataset.
  2. Data visualization to analyze correlations and perform descriptive statistics.
  3. Prepare data before model training.
  4. Train and evaluate multiple linear regression, decision tree regressor, random forest and XGBoost algorithms.
  5. Compare models results.

Action

Coding is available here

Results

After evaluating multiple machine learning models and comparing regression KPIs, XGBoost algorithm outperformed all the models in its predicted y values. An accuracy of 91% was achieved after training the model. This means that XGBoost model can describe 91% of the variability of the predicted variable.

Key Takeways / Lessons learned

This was a good project in which I gained a lot of knowledge and could put into practice my machine learning skills by evaluating multiple models. XGBoost is a powerful machine learning algorithm that can be used for regression tasks like this one. It has an advantage over other algorithms in terms of speed and memory utilization. This model can be easily implemented by used car dealers for their purchasing process.
