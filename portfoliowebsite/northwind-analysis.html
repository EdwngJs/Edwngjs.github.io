Northwind Analysis

Dataset

Northwind Traders is a sample database used by Microsoft to demonstrate the features of some of its products. The database contains the sales data for Northwind Traders, a fictitious speciality foods export/import company.

Objective

The objective of this project was to perform analysis on the Northwind dataset to identify the top 10 products, analyze profitability, and determine the top employees.

Task

  1. Retrieve data from the Northwind database using SQL.
  2. Analyze the data using Python to extract insights on top products, profitability, and employee performance.
  3. Create visualizations to present the findings effectively.

Action

Data Retrieval: Utilize SQL queries on Python using sqlite3 library to extract relevant data from the Northwind database, including product information, order details, employee records, and financial data.

Analysis with Python: Leverage the sqlite3 Python library to analyze the retrieved data using SQL queries. Calculated profitability by aggregating revenue and cost data for each product. Identified the top 10 products based on sales volume or revenue. Analyzed employee performance metrics such as sales volume, revenue generated, or customer satisfaction ratings.

Visualization: Created visualizations using python library matplotlib to plot the analysis findings. This included bar charts displaying the top 10 products, profitability analysis, and employee performance metrics.

Results

  • The analysis revealed the top 10 products based on sales volume or revenue, providing insights into popular products within the Northwind dataset.
  • Profitability analysis highlighted the most profitable products or product categories, aiding in decision-making regarding product pricing, promotion, or inventory management.
  • Examination of employee performance metrics identified top-performing employees based on sales performance, customer satisfaction, or other relevant metrics.

  • Utilizing SQL and Python together allows for comprehensive data analysis and insights generation.
